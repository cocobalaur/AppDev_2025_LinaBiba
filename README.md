# Budget Tracker App

## Description  
The budget app is a C# application with abilities to review your yearly spending. 
It allows the user to search for specific items, filtering and different viewings such as a  
pie chart or a data grid. 

## Purpose / Context  
This project was created as part of the AppDev course in 2025. 
Throughout the semester, we learned how to design and develop a fully functional application in a collaborative team setting. 
The project involved planning, implementing and continuously updating the app with new features. 
This project strengthened our in skills in C#, user interface design
and project management, while also highlighting the importance of teamwork.
